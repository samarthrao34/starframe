Accessibility Statement

Last updated: August 13, 2025

1. Our Commitment

StarFrame Animation Studio is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

2. Accessibility Standards

We strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

3. Measures Taken

We have implemented various measures to ensure accessibility, including:

  • Semantic HTML: Using proper HTML elements and structures for screen readers
  • Keyboard Navigation: All interactive elements are accessible via keyboard
  • Alt Text: Descriptive alternative text for all meaningful images
  • Color Contrast: Sufficient color contrast ratios for text readability
  • Focus Indicators: Visible focus indicators for keyboard navigation
  • Aria Labels: ARIA labels for enhanced screen reader compatibility
  • Responsive Design: Mobile-friendly design that works with assistive technologies

4. Accessibility Features

4.1 Visual Accessibility

  • High contrast color schemes
  • Scalable fonts and flexible layouts
  • Clear visual hierarchy and spacing
  • Text alternatives for visual content

4.2 Motor Accessibility

  • Large clickable areas for buttons and links
  • Keyboard-only navigation support
  • No time-sensitive interactions
  • Drag-and-drop alternatives

4.3 Cognitive Accessibility

  • Clear and simple language
  • Consistent navigation and layout
  • Error prevention and clear error messages
  • Logical content structure

5. Assistive Technologies

Our website is designed to work with various assistive technologies, including:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Voice recognition software
  • Keyboard-only navigation
  • Screen magnification software
  • Switch navigation devices

6. Third-Party Content

While we strive to ensure accessibility across our website, some third-party content may not fully conform to accessibility standards. We are working with our partners to improve accessibility in these areas.

7. Mobile Accessibility

Our mobile experience includes:

  • Touch-friendly interface with appropriate target sizes
  • Screen reader compatibility on mobile devices
  • Responsive design that adapts to different screen sizes
  • Voice control and switch control support

8. Known Limitations

We are aware of the following accessibility limitations and are working to address them:

  • Some portfolio videos may not have captions (we are adding them progressively)
  • Complex animations in the hero section may not be fully accessible to all users
  • Some third-party embedded content may have accessibility gaps

10. Alternative Formats

If you need information from this website in an alternative format, we can provide:

  • Large print documents
  • Audio descriptions for visual content
  • Plain text versions of complex documents
  • Sign language interpretation for video content (upon request)

11. Assessment and Testing

We regularly assess the accessibility of our website through:

  • Automated accessibility testing tools
  • Manual testing with screen readers and other assistive technologies
  • User testing with people who have disabilities
  • Regular accessibility audits

12. Training and Awareness

Our team receives regular training on:

  • Web accessibility best practices
  • Inclusive design principles
  • Assistive technology usage
  • Legal requirements and standards

13. Ongoing Improvements

We are continuously working to improve accessibility by:

  • Regular accessibility reviews and updates
  • Implementing user feedback
  • Staying current with accessibility standards
  • Testing with real users who have disabilities

14. Legal Compliance

We strive to comply with applicable accessibility laws and regulations, including but not limited to:

  • Americans with Disabilities Act (ADA)
  • Section 508 of the Rehabilitation Act
  • European Accessibility Act (2019)
  • Rights of Persons with Disabilities Act (RPWD), 2016 - India
  • Web Content Accessibility Guidelines (WCAG) 2.1 Level AA
  • Information Technology Act, 2000 (amended 2008) - India
