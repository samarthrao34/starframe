## 🛠️ For You (Admin)

### Managing Shop Products
**To add new products**, edit `shop.html`:
1. Copy an existing product card
2. Update image source
3. Change title, description, price
4. Update category and data attributes
5. Update the addToCart() function parameters

### Updating Prices
**Pricing Guide**: Edit `pricing-guide.html`
- Update prices in tier cards
- Add new example projects
- Adjust USD conversions as needed

**Shop**: Edit product cards in `shop.html`
- Update ₹ INR prices
- Recalculate ~$ USD equivalents

### International Rates
**Currency conversions**: Update in multiple files
- `commission.html` - International section
- `pricing-guide.html` - Price conversions
- `shop.html` - Product prices

Current rate used: ₹1 = ~$0.012 (₹83 = $1)
